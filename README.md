# Stage Monitoring Tool

## 📌 Projectbeschrijving
Dit project is een webapplicatie die het volledige stageproces ondersteunt en opvolgt.  
Van het indienen van een stagevoorstel tot de finale evaluatie.

## 🎯 Doel
Het doel van deze applicatie is om:
- Het stageproces te digitaliseren
- Communicatie te verbeteren tussen student, docent en bedrijf
- Evaluaties en opvolging te centraliseren

## 👥 Doelgroepen
- Studenten
- Stagecommissie
- Docenten
- Stagementoren
- Administratie

## ⚙️ Functionaliteiten
- Stagevoorstel indienen
- Goedkeuring/afkeuring door stagecommissie
- Beheer van stageovereenkomst
- Wekelijkse logboeken
- Evaluaties en feedback
- Eindoverzicht per student
